Step 1: Install Python 3.11
Download Python 3.11
During install check тАЬAdd Python to PATHтАЭ
Verify:
# python --version
-> It should show 3.11.x


Step 2: Install Required Libraries
Open PowerShell inside your project folder:
# pip install vosk sounddevice pyttsx3 pyaudio

If pyaudio fails:
# pip install pipwin
# pipwin install pyaudio


Step 3: Download Hindi Vosk Model
Go to:
https://alphacephei.com/vosk/models
Download:
vosk-model-small-hi-0.22
-> Extract it.
Place it in the same folder as your Python file.

Folder structure must be:
project_folder/

Step 4: Check Microphone
Run this:
# import sounddevice as sd
# print(sd.query_devices())
Make sure your mic appears in the list.


Step 5: (If Voice Not Speaking) Install eSpeak
Download and install eSpeak NG.
Make sure it is added to PATH.


Step 6: Run the Program Using Command Prompt (CMD)
Press Windows + R.
Type cmd and press Enter.
Navigate to your project folder using the cd command.
Example:

# cd Desktop\project_folder
(Replace project_folder with your actual folder name.)

Once inside the project directory, run the following command:
>>>python hindi_assistant.py
